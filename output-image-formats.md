# Formati izlaznih slika

### TIFF (16-bitni)

16-bitni (digitalni broj piksela 0 - 65535) izlazni format TIFF (.tif).

### TIFF (32-bitni, postotak %)

32-bitni (pixel float 0,0 - 1,0) TIFF (.tif) format izlaza. Vrijednosti piksela predstavljaju postotke refleksije gdje je 0,0 = 0% i 1,0 = 100%.

### **JPG (8-bitni)**

8-bitni (digitalni broj piksela 0 -255) izlazni format JPG (.jpg).

### **PNG (8-bitni)**

8-bitni (digitalni broj piksela 0 - 255) PNG (.png) format izlaza. Kompresija bez gubitaka prikladna za web pregled i vizualizaciju.
